# Age of Empires II: Random Map Scripts

A Visual Studio Code extension providing syntax highlighting for [Age of Empires II](https://www.ageofempires.com/) [random map scripts](https://docs.google.com/document/d/1jnhZXoeL9mkRUJxcGlKnO98fIwFKStP_OBozpr0CHXo) (`.rms`).

## Features

- Syntax highlighting for `.rms` files
- Nested block comments with arbitrary depth
- Bracket matching and block comment toggling (`Shift+Alt+A`)
- Support for assigning constants to other constants
- Highlighting of operators, numbers, and constants in math expressions

## Related Projects

- [tree-sitter-aoe2-rms](https://github.com/twestura/tree-sitter-aoe2-rms) — Tree-sitter grammar for editors with Tree-sitter support (Neovim, Emacs, Zed)
- [zed-aoe2-rms](https://github.com/twestura/zed-aoe2-rms) — Zed extension
